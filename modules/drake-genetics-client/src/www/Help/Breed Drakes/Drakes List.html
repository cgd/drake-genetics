Drakes List

The "Drakes" listing on the left hand side of the interface is a simple expandable list with two categories "Females" and "Males". By clicking on the right facing arrow control next to the folder icon by each category, you will expand the list with the set of female or male drakes available for breeding.

Each list is initially seeded with some example drakes.
Females:

  • P1 - Steel scale, gold eyes
  • PAT - Sand scale, white eyes
  • DB - Charcoal scale, red eyes
  • Scooby - Gold scale, gold eyes, scruffy
Males:
  • P2 - Sand scale, white eyes
  • Shaggy - Sand scale, white eyes, scruffy
Usage

Usage is fairly straight forward. Selecting either a female or male will cause this drake to appear both in the appropriate position in the breeding pair (female on the left, male on the right), and in the detail panel at the bottom of the window. You can freely change the breeding pair with no affect until you choose breed.

New drakes can be added to the list via the detail panel.
